Textualize performs indepth analysis of any given text.The analysis includes : Spam Vs Ham Classification, Hate/Abuse Speech Detection and Sentiment Analysis.

Each prediction page is connected with a Machine Learning Model which uses either of Logistic Regression or Multinomial Naive Bayes Algorithm to predict the results.
